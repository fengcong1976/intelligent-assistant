## 系统要求

- Python 3.10+
- Windows 10/11
- 8GB+ RAM

## 安装说明

1. 解压发布包
2. 安装依赖：`pip install -r requirements.txt`
3. 配置环境变量：复制 `.env.example` 为 `.env` 并填写配置
4. 运行程序：`python run_gui.py`

## 配置说明

### LLM配置
需要配置至少一个LLM服务：
- 智谱AI：填写 `ZHIPUAI_API_KEY`
- 通义千问：填写 `DASHSCOPE_API_KEY`
- OpenAI：填写 `OPENAI_API_KEY`

### 邮箱配置（可选）
如需使用邮件功能，填写：
- `AGENT_EMAIL`: 邮箱地址
- `AGENT_EMAIL_PASSWORD`: 邮箱密码或授权码
- `AGENT_EMAIL_SMTP`: SMTP服务器
- `AGENT_EMAIL_PORT`: SMTP端口

### 天气配置（可选）
如需使用天气功能，填写：
- `QWEATHER_API_KEY`: 和风天气API密钥

### 网易云音乐配置（可选）
如需使用音乐功能，确保：
- 网易云音乐API服务已启动
- 配置 `NETEASE_API_URL`

### HomeAssistant配置（可选）
如需使用智能家居功能，填写：
- `HOMEASSISTANT_URL`: HomeAssistant地址
- `HOMEASSISTANT_TOKEN`: 访问令牌

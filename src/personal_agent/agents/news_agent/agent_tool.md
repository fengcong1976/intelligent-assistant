# 新闻获取工具

## 功能说明
本工具用于获取新闻资讯，支持以下功能：
- 获取最新新闻
- 按分类获取新闻（科技、财经、综合等）
- 搜索新闻（按关键词）
- 获取热点新闻
- 支持多个新闻源

## 参数说明

### fetch_news（获取新闻）
- `count`: 新闻数量
  - 类型：integer
  - 示例：10
  - 说明：可选参数，默认10条

- `category`: 新闻分类
  - 类型：string
  - 示例："科技"、"财经"、"综合"
  - 说明：可选参数，默认综合

- `source`: 新闻来源
  - 类型：string
  - 示例："36kr"、"虎嗅"
  - 说明：可选参数，默认多个来源

### fetch_hot（获取热点）
- `count`: 热点数量
  - 类型：integer
  - 示例：10
  - 说明：可选参数，默认10条

### search_news（搜索新闻）
- `keyword`: 搜索关键词
  - 类型：string
  - 示例："AI"、"人工智能"
  - 说明：必需参数

- `count`: 搜索结果数量
  - 类型：integer
  - 示例：10
  - 说明：可选参数，默认10条

## 使用场景

### 场景1：获取最新新闻
```
用户："今天有什么新闻"
调用：fetch_news()
```

### 场景2：获取特定分类新闻
```
用户："科技新闻"
调用：fetch_news(category="科技")
```

### 场景3：获取热点新闻
```
用户："获取热点"
调用：fetch_hot()
```

### 场景4：搜索新闻
```
用户："搜索AI相关的新闻"
调用：search_news(keyword="AI")
```

### 场景5：获取特定来源新闻
```
用户："获取36kr的新闻"
调用：fetch_news(source="36kr")
```

## 返回信息

### fetch_news返回
- 新闻标题
- 新闻摘要
- 新闻链接
- 发布时间
- 新闻来源
- 新闻分类

### fetch_hot返回
- 热点新闻标题
- 热点新闻摘要
- 热点新闻链接
- 热度指数

### search_news返回
- 搜索结果标题
- 搜索结果摘要
- 搜索结果链接
- 相关度评分

## 注意事项

### 1. 新闻分类
- 支持的分类：科技、财经、综合、体育、娱乐、国际、国内
- 默认分类：综合
- 不支持的分类会提示用户

### 2. 新闻来源
- 支持多个新闻源
- 默认从多个来源获取
- 可以指定特定来源
- 新闻源不可用时会尝试其他来源

### 3. 搜索功能
- 支持关键词搜索
- 搜索结果按相关度排序
- 可以搜索历史新闻

### 4. 网络问题
- 网络超时会提示用户
- 网络不可用时会提示用户检查网络
- 建议稍后重试

### 5. 新闻时效性
- 获取的是最新新闻
- 热点新闻实时更新
- 搜索结果包含最新和历史新闻

## 常见问题

### Q1：可以获取哪些分类的新闻？
A：支持科技、财经、综合、体育、娱乐、国际、国内等分类。

### Q2：新闻来源有哪些？
A：支持多个新闻源，包括36kr、虎嗅、新浪、腾讯等。

### Q3：搜索功能支持历史新闻吗？
A：支持，搜索结果包含最新和历史新闻。

### Q4：新闻获取失败怎么办？
A：检查网络连接，稍后重试，或尝试其他新闻源。

## Description

这个技能用于播放音乐。支持播放本地音乐库中的歌曲，也支持搜索和播放在线音乐。可以按歌手、歌曲名、专辑等方式搜索。

## Help

🎵 **音乐智能体** - 播放和控制音乐

**播放控制：**
- 播放 / 播放音乐 / 播放 [歌曲名]
- 暂停 / 暂停播放
- 停止 / 停止播放
- 继续 / 继续播放
- 下一首 / 上一首

**音量控制（播放器音量）：**
- 静音 / 取消静音
- 声音大一点 / 声音小一点

**其他：**
- 搜索 [歌曲名]
- 播放列表

💡 提示：不带@时，静音/音量控制使用系统音量

## When to use

- 用户说"播放XXX的歌"
- 用户说"我想听音乐"
- 用户说"放首歌"
- 用户提到"音乐"、"歌曲"、"播放"等关键词
- 用户想暂停、继续、切歌

## How to use

1. 解析用户请求：
   - 提取歌手名、歌曲名、专辑名等信息
   - 判断是播放特定歌曲还是随机播放
   
2. 搜索音乐：
   - 首先在本地音乐库搜索
   - 如果本地没有，可以搜索网易云音乐
   
3. 播放控制：
   - 播放：action=play
   - 暂停：action=pause
   - 继续：action=resume
   - 下一首：action=next
   - 上一首：action=previous
   - 调节音量：action=volume

## Edge cases

- 找不到歌曲时，告知用户并推荐相似歌曲
- 音乐库为空时，引导用户添加音乐或播放在线音乐
- 正在播放时用户要求播放新歌，先停止当前播放
- 用户说"播放音乐"但没有指定歌曲时，随机播放

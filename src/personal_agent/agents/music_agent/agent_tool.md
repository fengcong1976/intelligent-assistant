# 音乐播放工具

## 功能说明
本工具用于播放和控制音乐，支持以下功能：
- 播放本地音乐库中的歌曲
- 搜索和播放在线音乐（网易云音乐）
- 播放控制（播放、暂停、继续、上一首、下一首）
- 音量控制（播放器音量）
- 按歌手、歌曲名、专辑搜索
- 查看播放列表

## 参数说明

### play（播放音乐）
- `song`: 歌曲名
  - 类型：string
  - 示例："晴天"、"告白气球"
  - 说明：可选参数

- `artist`: 歌手名
  - 类型：string
  - 示例："周杰伦"、"陈奕迅"
  - 说明：可选参数

- `album`: 专辑名
  - 类型：string
  - 示例："叶惠美"
  - 说明：可选参数

- `source`: 音乐来源
  - 类型：string
  - 示例："local"、"netease"
  - 说明：可选参数，默认先搜索本地，再搜索在线

### pause（暂停播放）
- 无参数

### resume（继续播放）
- 无参数

### next（下一首）
- 无参数

### previous（上一首）
- 无参数

### volume（音量控制）
- `action`: 音量操作
  - 类型：string
  - 示例："up"、"down"、"mute"、"unmute"
  - 说明：必需参数

- `value`: 音量值
  - 类型：integer
  - 示例：50
  - 说明：可选参数，0-100

### search（搜索音乐）
- `keyword`: 搜索关键词
  - 类型：string
  - 示例："周杰伦"
  - 说明：必需参数

- `source`: 音乐来源
  - 类型：string
  - 示例："local"、"netease"
  - 说明：可选参数，默认搜索所有来源

### show_playlist（显示播放列表）
- 无参数

## 使用场景

### 场景1：播放特定歌曲
```
用户："播放晴天"
调用：play(song="晴天")
```

### 场景2：播放歌手的歌
```
用户："播放周杰伦的歌"
调用：play(artist="周杰伦")
```

### 场景3：播放控制
```
用户："暂停"
调用：pause()

用户："继续"
调用：resume()

用户："下一首"
调用：next()
```

### 场景4：音量控制
```
用户："声音大一点"
调用：volume(action="up")

用户："静音"
调用：volume(action="mute")
```

### 场景5：搜索音乐
```
用户："搜索周杰伦"
调用：search(keyword="周杰伦")
```

### 场景6：查看播放列表
```
用户："播放列表"
调用：show_playlist()
```

## 注意事项

### 1. 音乐来源
- 本地音乐库：优先搜索本地
- 在线音乐：网易云音乐
- 默认先搜索本地，找不到再搜索在线

### 2. 播放控制
- 播放：开始播放指定歌曲
- 暂停：暂停当前播放
- 继续：恢复暂停的播放
- 上一首/下一首：切换歌曲

### 3. 音量控制
- 音量控制针对播放器音量
- 不带@时，静音/音量控制使用系统音量
- 音量范围：0-100

### 4. 搜索功能
- 支持按歌曲名搜索
- 支持按歌手名搜索
- 支持按专辑名搜索
- 支持模糊搜索

### 5. 播放列表
- 显示当前播放列表
- 显示当前播放歌曲
- 显示播放状态（播放/暂停）

## 常见问题

### Q1：如何添加本地音乐？
A：将音乐文件放入本地音乐库目录，系统会自动扫描。

### Q2：可以播放在线音乐吗？
A：支持搜索和播放在线音乐（网易云音乐）。

### Q3：找不到歌曲怎么办？
A：系统会推荐相似歌曲，或尝试其他搜索关键词。

### Q4：如何创建播放列表？
A：当前版本不支持创建播放列表，后续版本会添加。

### Q5：音乐播放失败怎么办？
A：检查音乐文件是否存在，或检查网络连接（在线音乐）。

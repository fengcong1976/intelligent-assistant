# 视频播放工具

## 功能说明
本工具用于播放和控制视频，支持以下功能：
- 播放本地视频文件
- 播放在线视频链接
- 播放控制（播放、暂停、继续、停止、上一首、下一首）
- 搜索视频
- 调节播放进度
- 调节播放速度
- 全屏控制

## 参数说明

### play（播放视频）
- `video`: 视频名称或路径
  - 类型：string
  - 示例："movie.mp4"、"https://example.com/video.mp4"
  - 说明：必需参数

- `source`: 视频来源
  - 类型：string
  - 示例："local"、"online"
  - 说明：可选参数，默认自动判断

### pause（暂停播放）
- 无参数

### resume（继续播放）
- 无参数

### stop（停止播放）
- 无参数

### next（下一个视频）
- 无参数

### previous（上一个视频）
- 无参数

### seek（调节播放进度）
- `position`: 播放位置（秒）
  - 类型：integer
  - 示例：60
  - 说明：必需参数

### speed（调节播放速度）
- `value`: 播放速度
  - 类型：number
  - 示例：1.5
  - 说明：必需参数，范围0.5-2.0

### fullscreen（全屏控制）
- `action`: 全屏操作
  - 类型：string
  - 示例："enter"、"exit"
  - 说明：必需参数

### search_video（搜索视频）
- `keyword`: 搜索关键词
  - 类型：string
  - 示例："电影"、"教程"
  - 说明：必需参数

### show_playlist（显示播放列表）
- 无参数

## 使用场景

### 场景1：播放本地视频
```
用户："播放movie.mp4"
调用：play(video="movie.mp4", source="local")
```

### 场景2：播放在线视频
```
用户："播放https://example.com/video.mp4"
调用：play(video="https://example.com/video.mp4", source="online")
```

### 场景3：播放控制
```
用户："暂停"
调用：pause()

用户："继续"
调用：resume()

用户："停止"
调用：stop()
```

### 场景4：切换视频
```
用户："下一个视频"
调用：next()

用户："上一个视频"
调用：previous()
```

### 场景5：调节进度
```
用户："跳到1分钟"
调用：seek(position=60)
```

### 场景6：调节速度
```
用户："1.5倍速播放"
调用：speed(value=1.5)
```

### 场景7：全屏控制
```
用户："全屏"
调用：fullscreen(action="enter")

用户："退出全屏"
调用：fullscreen(action="exit")
```

### 场景8：搜索视频
```
用户："搜索电影"
调用：search_video(keyword="电影")
```

### 场景9：显示播放列表
```
用户："播放列表"
调用：show_playlist()
```

## 注意事项

### 1. 视频来源
- 本地视频：支持多种格式（MP4、AVI、MKV等）
- 在线视频：支持常见视频网站链接
- 自动判断视频来源

### 2. 播放控制
- 播放：开始播放指定视频
- 暂停：暂停当前播放
- 继续：恢复暂停的播放
- 停止：停止当前播放

### 3. 播放进度
- 进度以秒为单位
- 可以跳转到任意位置
- 支持快进快退

### 4. 播放速度
- 速度范围：0.5-2.0
- 1.0为正常速度
- 大于1.0为快放，小于1.0为慢放

### 5. 全屏控制
- 支持进入和退出全屏
- 全屏模式下可以按ESC退出
- 不同播放器全屏方式可能不同

### 6. 视频格式
- 支持常见视频格式
- 不支持的格式会提示用户
- 建议使用MP4格式

## 常见问题

### Q1：可以播放哪些格式的视频？
A：支持常见视频格式，包括MP4、AVI、MKV、MOV等。

### Q2：如何搜索视频？
A：使用search_video，指定搜索关键词即可。

### Q3：播放速度可以调节吗？
A：可以，支持0.5-2.0倍速调节。

### Q4：视频播放失败怎么办？
A：检查视频文件是否存在，或检查网络连接（在线视频）。

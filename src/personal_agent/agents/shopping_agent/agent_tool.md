# 购物助手工具

## 功能说明
本工具用于管理购物相关的需求，支持以下功能：
- 商品搜索
- 商品比价
- 优惠信息获取
- 购物清单管理（创建、查看、添加、删除）
- 商品推荐

## 参数说明

### search_products（搜索商品）
- `keyword`: 商品关键词
  - 类型：string
  - 示例："iPhone"、"运动鞋"
  - 说明：必需参数

- `category`: 商品分类
  - 类型：string
  - 示例："电子产品"、"服装鞋包"
  - 说明：可选参数

### compare_prices（商品比价）
- `product_name`: 商品名称
  - 类型：string
  - 示例："AirPods"、"iPhone"
  - 说明：必需参数

### query_deals（获取优惠）
- `category`: 商品分类
  - 类型：string
  - 示例："电子产品"
  - 说明：可选参数

### create_list（创建购物清单）
- `name`: 清单名称
  - 类型：string
  - 示例："日常用品"、"生日礼物"
  - 说明：必需参数

### list_lists（查看购物清单）
- 无参数

### add_item（添加商品）
- `list_id`: 清单ID
  - 类型：string
  - 示例："list_123"
  - 说明：可选参数，默认使用第一个清单

- `name`: 商品名称
  - 类型：string
  - 示例："牛奶"
  - 说明：必需参数

- `price`: 商品价格
  - 类型：number
  - 示例：5.5
  - 说明：可选参数

- `quantity`: 商品数量
  - 类型：integer
  - 示例：5
  - 说明：可选参数，默认1

- `category`: 商品分类
  - 类型：string
  - 示例："食品"
  - 说明：可选参数

- `store`: 购买店铺
  - 类型：string
  - 示例："超市"
  - 说明：可选参数

- `url`: 商品链接
  - 类型：string
  - 示例："https://example.com/product"
  - 说明：可选参数

- `notes`: 备注信息
  - 类型：string
  - 示例："要新鲜的"
  - 说明：可选参数

### delete_item（删除商品）
- `list_id`: 清单ID
  - 类型：string
  - 示例："list_123"
  - 说明：必需参数

- `item_id`: 商品ID
  - 类型：string
  - 示例："item_456"
  - 说明：必需参数

### toggle_item（标记商品状态）
- `list_id`: 清单ID
  - 类型：string
  - 示例："list_123"
  - 说明：必需参数

- `item_id`: 商品ID
  - 类型：string
  - 示例："item_456"
  - 说明：必需参数

### recommend_products（推荐商品）
- `category`: 商品分类
  - 类型：string
  - 示例："电子产品"
  - 说明：可选参数

## 使用场景

### 场景1：搜索商品
```
用户："搜索iPhone"
调用：search_products(keyword="iPhone")
```

### 场景2：商品比价
```
用户："比价AirPods"
调用：compare_prices(product_name="AirPods")
```

### 场景3：获取优惠
```
用户："查看优惠"
调用：query_deals()
```

### 场景4：创建购物清单
```
用户："创建购物清单日常用品"
调用：create_list(name="日常用品")
```

### 场景5：添加商品
```
用户："添加牛奶到购物清单"
调用：add_item(name="牛奶")
```

### 场景6：查看购物清单
```
用户："我的购物清单"
调用：list_lists()
```

### 场景7：删除商品
```
用户："删除购物清单中的牛奶"
调用：delete_item(list_id="list_123", item_id="item_456")
```

### 场景8：推荐商品
```
用户："推荐商品"
调用：recommend_products()
```

## 注意事项

### 1. 商品搜索
- 支持多个电商平台
- 搜索结果按相关度排序
- 可以按分类筛选

### 2. 商品比价
- 对比多个平台价格
- 价格实时更新
- 显示价格趋势

### 3. 优惠信息
- 优惠信息来自官方渠道
- 注意优惠截止时间
- 部分优惠需要优惠券

### 4. 购物清单
- 支持多个清单
- 可以标记商品状态
- 支持添加备注信息

### 5. 商品推荐
- 基于用户历史推荐
- 支持按分类推荐
- 推荐仅供参考

## 常见问题

### Q1：可以搜索哪些电商平台的商品？
A：支持主流电商平台，包括淘宝、京东、拼多多等。

### Q2：比价功能准确吗？
A：价格实时更新，但可能有延迟，建议购买前确认。

### Q3：购物清单可以分享吗？
A：当前版本不支持分享，后续版本会添加分享功能。

### Q4：如何删除整个购物清单？
A：需要逐个删除商品，或使用delete_list功能（如果支持）。

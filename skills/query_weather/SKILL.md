## Description

这个技能用于查询天气信息。可以查询当前天气、未来几天的天气预报。支持查询任意城市的天气。

## When to use

- 用户说"XXX天气怎么样"
- 用户说"今天天气"
- 用户说"明天会下雨吗"
- 用户提到"天气"、"温度"、"下雨"等关键词
- 用户计划出行需要了解天气

## How to use

1. 识别城市：
   - 从用户请求中提取城市名
   - 如果用户没有指定城市，使用用户设置的默认城市
   
2. 调用天气智能体：
   - 调用 weather_agent 查询天气信息
   
3. 格式化返回：
   - 将天气信息以友好的方式呈现给用户
   - 包含温度、天气状况、湿度、风向等信息

## Edge cases

- 城市名不识别时，询问用户具体城市
- 天气查询失败时，告知用户稍后重试
- 用户问未来天气时，说明预报天数限制

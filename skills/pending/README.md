# 待开发技能目录

此目录存放系统自动检测到缺失技能后生成的 Skill 文档。

## 目录说明

当用户请求需要系统当前不具备的能力时，LLM 会自动分析并生成 Skill 文档到此目录。

## 处理流程

1. **自动检测** - LLM 分析用户请求，识别缺失的技能
2. **生成文档** - 自动创建 `xxx_agent.md` 文件
3. **人工审核** - 开发者查看并确认技能定义
4. **实现代码** - 创建对应的 `xxx_agent.py` 智能体
5. **移动文件** - 将 `.md` 文件移动到 `src/personal_agent/agents/` 目录
6. **注册智能体** - 在 `master.py` 中注册新智能体

## 文件格式

每个 `.md` 文件包含：
- 技能名称和描述
- 触发场景
- 支持的操作
- 实现建议

## 示例

```
skills/pending/
├── decrypt_agent.md    # 解密智能体（ncm/qmc等）
├── translator_agent.md # 翻译智能体
└── ocr_agent.md        # OCR文字识别智能体
```

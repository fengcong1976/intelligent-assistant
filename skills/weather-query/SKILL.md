# Weather Query Skill

## 1. Description
查询实时天气数据，支持街道级精确定位，数据来源和风天气。

**数据来源：**
- 和风天气 (qweather.com) - 专业天气数据服务
- 支持街道级精确定位（如"西安新城区韩森寨街道长兴路"）
- 数据权威可靠，覆盖全球城市

**支持的城市：**
- 中国：北京、上海、广州、深圳、杭州、南京、成都、重庆、武汉、西安等
- 全球：支持任何城市名称（中英文均可）
- 街道级：支持精确到街道的天气查询

## 2. When to use
- 用户说："今天天气"
- 用户说："北京天气怎么样"
- 用户说："上海现在多少度"
- 用户说："广州会下雨吗"
- 用户说："帮我查一下深圳的天气"
- 用户说："明天天气怎么样"

## 3. How to use
直接调用 execute(city="城市名") 即可，无需额外参数。

返回格式：
```
🌤️ 西安新城区韩森寨街道长兴路当前天气 (街道级) (16:36)

🌡️ 温度: 18°C (体感14°C)
☁️ 天气: 晴
💧 湿度: 0%
🌬️ 风向风速: 南风 6 km/h
👁️ 能见度: 16 km
📊 气压: 974 hPa

📍 数据来源: 和风天气
```

## 4. Edge cases
- 城市名称模糊：自动去除"市"、"省"后缀
- 不支持的城市：提示用户确认城市名称
- 网络超时：10秒超时保护

## 5. Performance
- 平均响应时间：1-2秒
- 数据更新频率：实时
- 支持街道级精确定位

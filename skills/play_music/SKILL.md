## Description

这个技能用于播放音乐。支持播放本地音乐库中的歌曲，也支持搜索和播放在线音乐。可以按歌手、歌曲名、专辑等方式搜索。

## When to use

- 用户说"播放XXX的歌"
- 用户说"我想听音乐"
- 用户说"放首歌"
- 用户提到"音乐"、"歌曲"、"播放"等关键词
- 用户想暂停、继续、切歌

## How to use

1. 解析用户请求：
   - 提取歌手名、歌曲名、专辑名等信息
   - 判断是播放特定歌曲还是随机播放
   
2. 搜索音乐：
   - 首先在本地音乐库搜索
   - 如果本地没有，可以搜索网易云音乐
   
3. 播放控制：
   - 播放：调用 music_agent 的 play 功能
   - 暂停：调用 pause 功能
   - 继续：调用 resume 功能
   - 下一首：调用 next 功能
   - 调节音量：调用 volume 功能

## Edge cases

- 找不到歌曲时，告知用户并推荐相似歌曲
- 音乐库为空时，引导用户添加音乐或播放在线音乐
- 正在播放时用户要求播放新歌，先停止当前播放
- 用户说"播放音乐"但没有指定歌曲时，随机播放
